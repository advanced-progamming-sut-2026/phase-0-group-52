# Plants vs. Zombies 2 — Advanced Programming Project

> An implementation of **Plants vs. Zombies 2**, developed for the Advanced Programming course project at Sharif University of Technology.

## About the Project

This repository contains our implementation of a simplified but extensible version of **Plants vs. Zombies 2**.

The project is designed to help us practice object-oriented programming, clean architecture, teamwork, Git workflows, testing, and maintainable Java development.

The project is developed incrementally across multiple phases. In the early phases, the game is controlled through terminal commands. Later phases expand the gameplay systems, add graphics, and complete the features.
